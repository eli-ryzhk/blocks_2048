# 2048 Merge Blocks

Puzzle game inspired by 2048 mechanics.

The player interacts with a grid of numbered blocks. When two neighboring blocks with the same value are selected, they merge into a new block with a higher number.

The goal is to keep merging blocks, create larger values, and achieve the highest score possible.

---

## How to Play

1. Click on a numbered block  
2. Select a neighboring block with the same value  
3. The blocks merge into a new block with a larger number  
4. Continue combining blocks to create higher values  

---

## Controls

- **Left Mouse Button** — Select and merge blocks
